`.md` または `.markdown` ファイルをブラウザで開くと，自動でプレビューに切り替わります．ローカルファイルを使う場合は，ブラウザの拡張機能設定で「ファイルの URL へのアクセスを許可」を有効にしてください．

右下のツールバーから，ソース表示，コピー，印刷，読み上げ，エクスポート，設定を操作できます．設定では，テーマ，フォント，最大横幅，コードブロックの見た目，画像の高さを調整できます．

見出しがあるファイルでは目次が表示されます．画面が狭い場合はミニインジケーターに切り替わり，クリックすると目次を開けます．

最近開いたファイルは設定パネルから確認できます．履歴はブラウザ内に保存され，いつでも消去できます．
