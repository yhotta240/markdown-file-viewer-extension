ローカルの Markdown ファイルを表示するには，Chrome でファイルアクセスを許可する必要があります．

1. 下のリンクから拡張機能の詳細ページを開きます．
2. 「ファイルの URL へのアクセスを許可する」をオンにします．
3. このページに戻り，「許可できたか確認する」を押します．
4. `.md` ファイルをブラウザにドラッグすると，プレビューが表示されます．
