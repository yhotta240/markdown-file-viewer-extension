Markdown File Viewer は，Markdown ファイルをブラウザ内で表示するための拡張機能です．Markdown の解析，目次生成，表示設定の反映，エクスポート処理は端末内で行います．

## 収集と外部送信

本拡張機能は，個人を特定する情報，Markdown ファイルの本文，閲覧内容，入力内容を収集しません．外部サーバーへの送信，広告配信，分析，トラッキングも行いません．

ユーザーのデータを販売せず，第三者に提供せず，主要機能と関係のない目的で利用しません．信用調査，融資，保険，雇用などの判断にも利用しません．

## ブラウザ内に保存する情報

表示設定，拡張機能の有効状態，最近開いたファイルの情報，動作ログを Chrome Storage に保存します．最近開いたファイルには，URL，タイトル，ファイル名，最終表示日時，ローカルまたは Web の区分が含まれます．履歴とログは，設定パネルから確認または削除できます．

## 権限の利用目的

`storage` 権限は，表示設定，履歴，ログをブラウザ内に保存するために使います．

`.md`，`.markdown` へのアクセス権限は，対象ファイルを検出し，プレビューを表示するために使います．ローカルファイルを表示するには，ブラウザの拡張機能設定で「ファイルの URL へのアクセスを許可」を有効にする必要があります．

読み上げ機能は，ブラウザが提供する Web Speech API を使います．音声合成の処理方法は，利用しているブラウザや OS の仕様に従います．
