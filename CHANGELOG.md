# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

## [0.1.0] - 2026-06-06

### Added

- ツールバーにファイル情報ポップオーバーを追加
- コードブロックに折り返しボタンを追加
- コードブロックのテーマ選択とシンタックスハイライトを追加
- 表示設定にプレビュー画像の最大高さ調整を追加

### Fixed

- Markdown 表示時に raw 表示が一瞬見えるちらつきを抑制
- エクスポート失敗時にエラーログを出力

## [0.0.5] - 2026-06-06

### Changed

- ロギング処理と安定性を改善
- 拡張機能名を manifest から動的に表示するように変更

### Fixed

- 不要な通知機能を削除
- favicon が表示されない問題を修正

## [0.0.4] - 2026-06-05

### Added

- コードブロックにコピー機能を追加

### Fixed

- SVG アイコンをアイコンフォントに置き換え
- 目次の見出し検知範囲を拡大して判定精度を改善
- 目次の表示位置を修正

## [0.0.3] - 2026-06-04

### Added

- 目次のポップアップ機能を追加
- テキスト読み上げ機能を追加
- 読み上げ中のハイライトと関連機能を追加
- 読み上げ設定を追加

## [0.0.2] - 2026-06-04

### Added

- エクスポート機能を追加
- 印刷ボタンを追加
- ドキュメントの動的注入機能を追加
- 設定機能を強化
- 利用方法・ショートカット・プライバシー関連ドキュメントを追加

### Documentation

- README を更新

## [0.0.1] - 2026-06-03

### Added

- Markdown プレビュー機能を実装
- Chrome 拡張機能の初期構成を追加
